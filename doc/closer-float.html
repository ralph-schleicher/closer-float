CLOSER-FLOAT

Closer to floating-point arithmetic.

Provide a common interface to IEEE 754 (a.k.a. ISO/IEC/IEEE 60559)
and ISO/IEC 10967 features.

If a symbol is not bound, then the corresponding feature is not
available.
